# Task Clipboard

Windows向けの Electron + Vue アプリです。  
複数画像の一時クリップボード機能と、シンプルなタスク管理機能を1つにまとめています。

## 機能
- `クリップボード` タブ
- `Ctrl+V` でクリップボード画像を追加
- 画像ファイルのドラッグ＆ドロップ追加
- 画像プレビュー表示
- 画像の個別削除
- 画像保持上限 50 枚（超過時は最古を自動削除）
- 画像はアプリ起動中のみ保持（終了時に破棄）

- `タスク` タブ
- タスク追加
- チェック時に即時削除
- タスク一覧はローカル保存され再起動後も復元

- `最前面ピン止め`
- ヘッダー右上トグルで ON/OFF
- 状態を保存し、再起動時に復元（初回は OFF）

## 開発
```bash
npm install
npm run dev
```

## ビルド
```bash
npm run build
```

## Windows インストーラー生成
```bash
npm run dist:win
```
